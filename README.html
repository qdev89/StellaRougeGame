Stellar Rogue - How to Run the Game

Important: The game cannot be run directly from the file system due to browser security restrictions (CORS policy). You need to serve the files through a web server.

Option 1: Using Python's Built-in HTTP Server

If you have Python installed, you can use its built-in HTTP server:

For Python 3:

python -m http.server 8000

For Python 2:

python -m SimpleHTTPServer 8000

Then open your browser and navigate to: http://localhost:8000

Option 2: Using Visual Studio Code Live Server

If you have Visual Studio Code installed:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

Option 3: Using Node.js http-server

If you have Node.js installed:

# Install http-server globally
npm install -g http-server

# Run the server
http-server -p 8000

Then open your browser and navigate to: http://localhost:8000

Option 4: Using XAMPP, WAMP, or MAMP

If you have XAMPP, WAMP, or MAMP installed:

  1. Copy the game files to your htdocs or www directory
  2. Start the Apache server
  3. Navigate to http://localhost/your-game-folder

Note: The game has been modified to handle missing assets gracefully. If you see placeholder graphics or don't hear sound effects, this is expected behavior when running without a proper server.

Game Controls

  • Arrow Keys or WASD: Move the ship
  • Spacebar: Fire weapons
  • P: Pause the game
  • B: Toggle invincibility (debug mode)
